========= ===============
THICKDIFF R Documentation
========= ===============

Difference in Thickness
-----------------------

Description
~~~~~~~~~~~

Difference between the thickness of the grown silicon layer and its
target value.

Usage
~~~~~

::

   data(THICKDIFF)

Source
~~~~~~

E. Yashchin (1991)

Examples
~~~~~~~~

::

   data(THICKDIFF)

   plot(THICKDIFF, type="b")
