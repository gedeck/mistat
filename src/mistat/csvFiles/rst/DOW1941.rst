Dow-Jones Financial Index 1941
------------------------------

Description
~~~~~~~~~~~

The Dow-Jones daily index of 1941.

Usage
~~~~~

::

   data(DOW1941)

Source
~~~~~~

Kenett, R. and Zacks, S. (1998) *Modern Industrial Statistics: The
Design and Control of Quality and Reliability*. Duxbury Press.

Examples
~~~~~~~~

::

   data(DOW1941)

   plot(DOW1941,          
        type="b", 
        ylab="Dow Jones 1941")
