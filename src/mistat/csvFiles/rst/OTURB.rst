OTURB R Documentation

Cycle Times of a Piston
-----------------------

Description
~~~~~~~~~~~

100 cycle times (*s*) of a piston, as from ``pistonSimulation``.

Usage
~~~~~

::

   data(OTURB)

References
~~~~~~~~~~

See ``pistonSimulation``

Examples
~~~~~~~~

::

   data(OTURB)

   plot(OTURB, type="b")
